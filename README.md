# Dynamic Machine Learning Prediction Platform


#  Overview
This project is a web-based platform that allows users to build machine learning models without writing code.

# Features
- Upload CSV datasets
- Automatic preprocessing (encoding, scaling)
- Multiple ML models:
  - Decision Tree
  - Random Forest
  - Logistic Regression
  - KNN
- Model comparison
- Real-time prediction interface
- Download trained model

# Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas

# Run Locally
